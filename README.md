# VoltLogic 

VoltLogic is a web-based 2D Electronic Design Automation (EDA) tool designed for real-time digital logic simulation.

##  Live Demo
[voltlogic.vercel.app]

##  Key Features
* **Logic State Synchronization:** Engineered custom wire-logic to ensure instantaneous state propagation across complex gate networks.
* **AI-Collaborative Workflow:** Developed using an Agent-based workflow (referenced in AGENTS.md), leveraging advanced prompt engineering to architect the simulation engine.
* **Responsive UI:** Built with Next.js and optimized for performance-heavy 2D canvas interactions.

##  Tech Stack
* **Frontend:** Next.js, React, Tailwind CSS
* **Tools:** GitHub, Vercel, Claude/Opus (AI Debugging)
